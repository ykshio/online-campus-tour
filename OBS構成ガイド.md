### シーン4: スタジオ＋スライド

第1部の大学説明（PowerPoint）やスタジオつなぎの写真紹介で使用。
MCのWindows PCからHDMIキャプチャカードで映像を取り込む。

```
┌──────────────────────────┬──────────┐
│                          │          │
│   MC PC画面（HDMI経由）    │  カメラ   │
│  （PowerPoint/写真）       │  （小）   │
│                          │          │
│                          │          │
└──────────────────────────┴──────────┘
```

| ソース名 | 種類 | 設定 | 配置 |
|---------|------|------|------|
| MC PC画面 | 映像キャプチャデバイス | HDMIキャプチャカード | 左側メイン（約1440x1080） |
| カメラ | 映像キャプチャデバイス | SIGMA fpL（USB-C） | 右下（約480x270程度） |
| マイク | 音声入力キャプチャ | スタジオマイク | - |

> **接続方法**: MC PC [HDMI出力] → HDMIキャプチャカード [USB] → 配信PC
> MCがPowerPointを操作し、配信管理はOBSのシーン切り替えに専念する。
> MC PCではPowerPointを**スライドショーモード（全画面）**で表示すること。
> スタジオつなぎで写真を見せる場合も、MC PCで写真を開いて表示する。

## 当日の操作フロー

配信管理（草野・坂上）が行うシーン切り替えの流れ。

```
待機画面 ──→ タイトルカード ──→ スタジオ（OP挨拶）
                                   │
         ┌─────────────────────────┘
         ↓
    ===== 第1部：大学説明（MC PCのPowerPoint） =====

    ※ スライド操作はMC（森山）が手元のPCで行う
    ※ 配信管理はスタジオ ←→ スタジオ＋スライド の切り替えに専念

    スタジオ＋スライド ←→ スタジオ を適宜切り替え
    ├─ 注意事項・本日の流れ → スタジオ＋スライド
    ├─ 自己紹介 → スタジオ＋スライド
    ├─ 大学概要・アクセス → スタジオ＋スライド
    ├─ 学びの分野 → スタジオ＋スライド / 実体験はスタジオに切替
    ├─ キャンパスライフ → スタジオ＋スライド / 掛け合いはスタジオ
    ├─ 質問対応 → スタジオ＋質問テロップ ON
    ├─ 第2部導入 → スタジオ
         │
         ↓
    ===== 第2部：中継とスタジオの往復 =====

    中継（テロップ：1号館前 ON）
         ↓
    スタジオ（立地紹介トーク）
         ↓
    ⚠️ 2号館ここから：中継音声ミュート＋スタジオマイクON
    中継（テロップ：メディアセンター ON）← MC解説、利用者の顔映さない
         ↓
    スタジオ＋スライド（写真で補足）
         ↓
    中継（テロップ：一般教室 ON）← MC解説、利用者の顔映さない
         ↓
    ⚠️ 2号館ここまで：スタジオマイクミュート＋中継音声ON
    スタジオ（授業トーク）
         ↓
    中継（テロップ：学食 ON）
         ↓
    スタジオ＋スライド（メニュー写真）
         ↓
    中継（テロップ：ものづくりセンター ON）
         ↓
    スタジオ＋スライド（作例写真）
         ↓
    中継（テロップ：学習スペース ON）
         ↓
    中継（テロップ：ルーフガーデン ON）
         ↓
    ===== 第3部：座談会 =====

    スタジオ（トークライブ）
    ├─ テロップ：第3部 ON → OFF
    ├─ テロップ：安部/栗原/塩澤 順にON → OFF
    ├─ 質問表示 ON → OFF（繰り返し）
         │
         ↓
    スタジオ（ED）
         ↓
    タイトルカード ──→ 待機画面（配信終了）
```

## テロップ操作のコツ

- **場所テロップは常に1つだけ表示**。切り替え前に前のテロップをOFFにしてから次をONにする
- **人物テロップは話者紹介時のみ一時的にON**。紹介が終わったらOFFにする
- **セクションテロップはパート冒頭のみ**。数秒〜十数秒表示したらOFFにする
- **質問テロップ**は表示前にHTMLファイルを編集してテキストを書き換え、OBSでソースを「更新」してからONにする

## 2号館中継時の注意事項

2号館（メディアセンター・一般教室）は以下の条件を厳守すること。

1. **利用者の顔は映さない** → カメラ（栗原）は施設や設備を中心に撮影
2. **現地では声や音を一切出さない** → 安部は無言で施設内を案内。解説はスタジオのMCが行う

### 配信管理の操作手順（2号館中継時）
1. 中継シーンに切り替える前に、スタジオマイクを**ミュート解除**
2. 中継音声（VDO.Ninja）を**ミュート**
3. MCがスタジオから映像に合わせて解説
4. 2号館の中継が終わったら、スタジオマイクを**ミュート**に戻し、中継音声を**ミュート解除**

## 音声設定

| ソース | 対象シーン | 備考 |
|-------|----------|------|
| スタジオマイク | スタジオ / スタジオ＋スライド | 常時ON |
| スタジオマイク | 中継（2号館のみ） | 普段ミュート。2号館中継時のみONにしてMCが解説 |
| 中継音声（VDO.Ninja） | 中継 | ブラウザソースの音声出力を有効化。2号館中継時はミュート |
| BGM | スタジオ | 待機中やスタジオつなぎで薄く流す。トーク時は下げる |

> **Zoom出力**: OBSの全画面プロジェクター（外部ディスプレイ）をZoomで画面共有。
> 共有時に「コンピュータの音声を共有」にチェックを入れること。
